# SecureKey Generator 

SecureKey Generator is a Python-based application designed to create random and secure passwords according to user preferences. It helps users improve account security by generating strong passwords instantly.

## Features
- Generates random passwords
- User can set custom password length
- Supports uppercase and lowercase letters
- Includes numbers and special characters
- Simple and easy to use

## Technologies Used
- Python
- Random Module
- String Module
- Matplotlib (for password strength graph)

## Working Process
1. Enter the desired password length.
2. Choose the character types to include.
3. Generate the password.
4. Analyze password strength using a graph.

## Objective
The main objective of this project is to create strong and secure passwords for better protection of personal accounts and data.

## Conclusion
This project demonstrates how Python can be used to generate secure passwords and analyze their strength in a simple and effective way.
